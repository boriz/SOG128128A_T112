# SOG128128A_T112 Transparent OLED (1.12", 128x128)
This repository contains a small demo and reference docs for the Surenoo
SOG128128A_T112 transparent graphic OLED module (SH1107 controller).

The module is sold by [Surenoo](https://www.surenoo.com/products/detail/23109435) and is also commonly listed on AliExpress.

## Hardware overview
- Display: 1.12" transparent graphic OLED, 128x128 pixels
- Controller: SH1107 (or equivalent)
- Interface: SPI
- Logic supply: 3.3V
- Panel supply: Vpp = 15V
- Reference current: Iref = 16.625uA, Rref = 750K

## Repo contents
- `Bubbles/Bubbles.ino` Arduino demo (simple falling circles animation)
- `Documents` SOG128128A_T112 and SH1107 information
